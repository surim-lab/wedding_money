# 축의금 정산

브라우저에서 바로 여는 단일 페이지 축의금 정산 프로그램입니다.

## 실행

`index.html` 파일을 더블클릭해서 브라우저로 열면 됩니다.

## 기능

- 이름, 소속, 금액, 메모 입력
- 신랑 / 신부 구분 선택
- 선택한 구분에 따라 배경색 변경
- 소속 드롭다운 선택
- 총액, 인원, 평균 금액 자동 계산
- 50,000원부터 10원까지 권종별 현금 합계 계산
- 명단 총액과 실제 현금 합계 차이 표시
- 검색, 정렬, 수정, 삭제
- 브라우저 로컬 저장
- CSV 내보내기

중요한 정산 전에는 `CSV 내보내기`로 백업 파일을 만들어 두는 것을 권장합니다.
